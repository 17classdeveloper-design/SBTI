# SBTI

SBTI is a playful personality quiz app for iPhone. Answer 31 quick questions, discover one of 27 silly personality types, browse the ranking, and save a shareable result card.

# SBTI 中文说明

SBTI 是一款轻松、无厘头、适合分享的人格测试 App。回答 31 道快问快答，即可从 27 种趣味人格中找到你的类型，并保存结果卡片分享给朋友。
